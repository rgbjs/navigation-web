## 运行项目

**安装依赖:**

*npm*

```bash
npm i
```

*yarn*

```bash
 yarn install
```





**启动项目:**

*npm*

```bash
npm run serve
```

*yarn*

```bash
yarn run serve
```





**打包项目:**

*npm*

```bash
npm run build
```

*yarn*

```bash
yarn run build
```



## 注意事项

上线项目请删除 request 目录中的 baseURL
